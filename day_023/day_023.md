## 3. Market the hardware product
Product: `EV Battery Management System (EV BMS)`

Post idea:
An EV BMS provides the sensing accuracy needed to calculate battery state of charge, state of health, and remaining useful life.

Without reliable analog conditioning, the digital estimation algorithms have no foundation.

The hardware quality of the BMS directly determines how well the software can manage the pack.

## 4. Circuit scenario + SPICE
Scenario:
A multiple-feedback active anti-aliasing filter for ADC input.

Post idea:
Before an analog signal reaches the ADC in a BMS, high-frequency noise must be removed to prevent aliasing.

A practical circuit is a second-order multiple-feedback (MFB) low-pass filter built with an op-amp. SPICE can be used to verify the cutoff frequency, quality factor, passband gain, and roll-off slope.

The filter design must balance noise rejection against settling time so the ADC sampling accuracy is not compromised.

## 6. Analog IC datasheet analysis
Suggested IC: `LM358`

Technical idea:
The `LM358` can be evaluated as the active element in multiple-feedback filter stages for low-frequency BMS sensor conditioning.

Key datasheet parameters to analyze:
- Open-loop gain at low frequency
- Unity-gain bandwidth and gain-bandwidth tradeoff
- Output impedance and drive capability
- Input common-mode range vs. filter input swing
- Slew rate limitations on filter transient response
- Supply current for multi-channel filter banks

In an MFB filter, the op-amp gain-bandwidth product must be significantly higher than the filter cutoff to avoid Q-enhancement and passband distortion.
