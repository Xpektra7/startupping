## 4. Circuit scenario + SPICE
Scenario:
A thermocouple interface with cold-junction compensation for temperature sensing.

Post idea:
Some EV BMS applications require temperature measurement beyond the range of NTC thermistors, such as motor or inverter temperature monitoring.

A thermocouple interface uses an instrumentation amplifier and a cold-junction compensation sensor. SPICE can be used to study thermocouple voltage generation, cold-junction compensation accuracy, amplifier gain stability, and noise rejection.

## 6. Analog IC datasheet analysis
Suggested IC: `AD8495`

Technical idea:
The `AD8495` is a precision thermocouple amplifier with integrated cold-junction compensation, suitable for wide-range temperature monitoring in EV systems.

Key datasheet parameters to analyze:
- Thermocouple type compatibility (K, J, T)
- Cold-junction compensation accuracy over temperature
- Output scaling and offset
- Gain error and nonlinearity
- Noise rejection and filtering
- Supply voltage range and quiescent current

In an EV context, the AD8495 simplifies thermocouple interface design by providing a calibrated, pre-conditioned output that can be read directly by the BMS ADC.
