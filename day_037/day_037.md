## 4. Circuit scenario + SPICE
Scenario:
A redundant sensor voting circuit for safety-critical measurements.

Post idea:
Safety-critical BMS parameters such as overcurrent detection or cell overvoltage often require redundant sensors. A voting circuit compares multiple sensor outputs and takes action when two or more agree.

A practical circuit uses comparators and logic gates to implement 2-of-3 voting. SPICE can be used to verify the voting threshold, fault detection latency, and behavior under single-sensor failure.

## 6. Analog IC datasheet analysis
Suggested IC: `LM358`

Technical idea:
The `LM358` dual op-amp can be used in redundant sensor interface circuits where two independent amplifiers condition the same measurement signal.

Key datasheet parameters to analyze:
- Channel-to-channel isolation and crosstalk
- Offset voltage matching between channels
- Supply current for dual-channel operation
- Common-mode rejection matching
- Temperature drift matching
- Output drive capability per channel

In a redundant design, the matching between amplifier channels affects the ability to detect sensor faults without generating false positives.
