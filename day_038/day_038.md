## 4. Circuit scenario + SPICE
Scenario:
A watchdog timer circuit for BMS controller supervision.

Post idea:
The main microcontroller in an EV BMS must be monitored by an external watchdog to detect lockup or software faults.

A practical watchdog timer circuit uses a retriggerable monostable multivibrator that resets the controller if the watchdog input is not toggled within the timeout period. SPICE can be used to study the timeout accuracy, reset pulse width, and behavior during power-up and brownout conditions.

## 6. Analog IC datasheet analysis
Suggested IC: `TPL5010`

Technical idea:
The `TPL5010` is a nano-power watchdog timer IC that can be evaluated for supervising BMS microcontrollers in always-on automotive applications.

Key datasheet parameters to analyze:
- Watchdog timeout period accuracy and range
- Supply current in active and idle modes
- Reset output type and pulse width
- Enable/disable control and startup behavior
- Operating voltage range
- Temperature stability of timeout period

In a BMS watchdog application, the TPL5010 provides reliable supervision with extremely low power consumption, making it suitable for systems that must remain active even when the vehicle is parked.
