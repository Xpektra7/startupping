## 4. Circuit scenario + SPICE
Scenario:
A battery temperature sensing circuit for an EV BMS.

Post idea:
Temperature monitoring is a core safety function in any EV BMS because thermal runaway risk must be detected early.

A practical analog setup uses an NTC thermistor in a voltage divider, followed by buffering and filtering before ADC conversion. SPICE simulation can be used to study sensor linearity, divider behavior, sensor sensitivity, and response across different temperatures.

This type of circuit helps the BMS react before the battery enters an unsafe thermal state and gives the controller a reliable analog input.

## 6. Analog IC datasheet analysis
Suggested IC: `LM358`

Technical idea:
The `LM358` is a general-purpose dual op-amp that can be evaluated for buffering and conditioning low-speed sensor signals inside EV BMS support circuits.

Key datasheet parameters to analyze:
- Supply voltage range
- Input common-mode voltage range
- Output voltage swing
- Slew rate
- Unity-gain bandwidth
- Input offset voltage and bias current

In an EV BMS, these specifications matter when designing sensor interfaces that must remain stable, accurate, and compatible with battery-powered supply rails.

The main design concern is whether the op-amp can operate within the available headroom while preserving measurement accuracy at the ADC input.
