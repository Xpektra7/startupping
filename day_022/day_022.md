## 3. Market the hardware product
Product: `EV Battery Management System (EV BMS)`

Post idea:
The EV BMS is the decision-making layer that translates sensor data into battery protection actions.

It monitors voltage, current, and temperature, then uses threshold logic and control algorithms to keep the pack inside safe limits.

Every reliable electric vehicle depends on this feedback loop to ensure long battery life and safe operation.

## 4. Circuit scenario + SPICE
Scenario:
A level-shifting circuit for bipolar current sense signals.

Post idea:
In EV BMS current sensing, the shunt voltage can be bipolar (charging and discharging). Many ADC inputs are unipolar, so the signal must be level-shifted into the positive range.

A practical circuit uses an op-amp summing amplifier that adds a precision DC offset to the differential signal. SPICE can be used to verify the transfer function, offset accuracy, linear range, and common-mode rejection across the full operating range.

## 6. Analog IC datasheet analysis
Suggested IC: `TLV9002`

Technical idea:
The `TLV9002` is a low-voltage rail-to-rail op-amp suitable for level-shifting and scaling analog signals in battery-powered BMS designs.

Key datasheet parameters to analyze:
- Rail-to-rail input and output voltage range
- Offset voltage and temperature drift
- Gain bandwidth product
- Slew rate
- Input bias current
- Supply voltage range and quiescent current

In a level-shifting application, these parameters determine whether the amplifier maintains accuracy across the full signal swing while operating from a single supply rail.
