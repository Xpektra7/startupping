# Business & Revenue Model

## Monetization Strategy

Pure **B2B SaaS subscription**. Hardware is provisioned at-cost — recurring revenue is generated entirely from the monthly subscription fee per active camera feed.

### Pricing Tier

| Tier | Price | What's included |
|---|---|---|
| Single unit | $29/mo | 1 camera feed, SMS alerts, health telemetry |
| Ranch (up to 5) | $99/mo | 5 feeds, priority alert routing, web dashboard |
| Agency (unlimited) | $499/mo | Unlimited feeds, CRM integration, API access, SLA |

Hardware one-time cost (at-cost): ~$120 per unit (solar panel, battery, camera module, custom trigger board, enclosure). No markup.

---

## Unit Economics

| Item | Cost |
|---|---|
| BOM per unit (at volume) | ~$85 |
| Assembly + test | ~$25 |
| Shipping + deployment support | ~$10 |
| **Total hardware cost** | **~$120** (passed through at-cost) |
| Monthly subscription (single tier) | $29 |
| Monthly cloud + SMS cost | ~$3 |
| **Gross margin on subscription** | **~90%** |
| **Payback period on hardware** | **~4–5 months** |

---

## North Star Metric

**Verified Detections per Week.**

We measure startup value by the count of actionable, human-verified intrusion alerts delivered to end-users per week. This single metric captures:

- Detection accuracy (analog + AI combined)
- System uptime and reliability
- Real-world operational utility

Monthly churn <5% is the secondary benchmark — if the system isn't catching real events, users stop paying.

## Go-to-Market Motion

1. **Direct outreach** to private ranchers via county agricultural extension offices
2. **Pilot partnerships** with county sheriffs (ready-made pain, procurement pressure)
3. **Wildlife NGO referrals** for conservancy deployments
4. Early adopter pricing: first 6 months at 50% discount in exchange for field telemetry data to improve the analog trigger tuning
